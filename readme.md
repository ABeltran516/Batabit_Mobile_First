### Batabit Web App
> En este proyecto encontrarás una landing page estática sobre una aplicación web ficticia de intercambio de cryptomonedas. 

> Pretende ser un servicio que ofrezca información detallada sobre le valor de los activos, asi como la posibilidad de adquirirlos por medio de la misma plataforma.

Esta pagina está hecha con la finalidad de practicar el **diseño responsivo** de una pagina web, el uso de paletas colores adecuada, asi como el dominio técnico de aspectos como la **gestión de variables en CSS** y la **arquitectura BEM** para la organización de clases.
